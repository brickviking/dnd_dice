# Python DND Dice Roller
Roll up to 100 dice with sides of 2, 4, 6, 8, 10, 12, 20 and 100.

## Python2
python dnd.py numDice numSides

## Python3
python3 dnd.py numDice numSides

### Sample Output
python3 dnd.py 4 12  
4d12: 1:2 2:5 3:2 4:7 Total of 16/48
